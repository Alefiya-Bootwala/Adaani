================================================================================
KEY FEATURES
================================================================================

✅ Context-Only Prompting
   - LLM sees ONLY retrieved chunks
   - No full document injection
   - Prevents hallucination

✅ Mandatory Citations
   - Every answer includes [p5:c2] references
   - Traceable to source
   - Enables verification

✅ Smart Chunking
   - Semantic sentence-level splitting
   - Configurable overlap
   - Unique chunk IDs

✅ Retrieval Visibility
   - Shows top-5 retrieved chunks
   - Displays similarity scores
   - Debug output for every query

✅ Multi-turn Conversation
   - Maintains conversation history
   - Fresh retrieval per question
   - Follows up naturally

✅ "Not Found" Handling
   - Refuses to guess
   - Responds: "Not found in the document."
   - No external knowledge

✅ Persistent Index
   - Caches embeddings locally
   - Subsequent runs fast (~10 sec)
   - Use --no-cache to rebuild
