================================================================================
STEP 1: GET YOUR GEMINI API KEY
================================================================================

1. Go to: https://aistudio.google.com/app/apikey
2. Click "Get API Key"
3. Follow prompts to create API key
4. Copy your API key (looks like: AIzaSy...)

================================================================================
STEP 2: SET ENVIRONMENT VARIABLE
================================================================================

Open PowerShell and run:

$env:GOOGLE_API_KEY = "your-api-key-here"

Example:
$env:GOOGLE_API_KEY = "AIzaSyDxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"

================================================================================
STEP 3: INSTALL DEPENDENCIES
================================================================================

pip install -r requirements.txt

This installs:
  ✓ google-generativeai (Gemini API)
  ✓ pdfplumber (PDF processing)
  ✓ chromadb (Vector database)
  ✓ sentence-transformers (Embeddings)

================================================================================
STEP 4: RUN THE SYSTEM
================================================================================

Single document:
  python main.py ADANI.pdf

Multiple documents (recommended):
  python multi_document_qa.py

With tests:
  python main.py ADANI.pdf --test

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "GOOGLE_API_KEY not set"
Solution: $env:GOOGLE_API_KEY = "your-key"

Issue: API Key not working
Solution: Verify key at https://aistudio.google.com/app/apikey

Issue: Rate limits
Solution: Gemini has free tier limits, wait between queries

Issue: Connection error
Solution: Check internet connection, verify firewall allows Google API
